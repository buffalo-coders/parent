# org.buffalo-coders:parent

Apache Maven top level pom for all Buffalo Coders projects.

## Usage

Leverage this top level pom via `<parent>` tag:

```xml
  <parent>
    <groupId>org.buffalo-coders</groupId>
    <artifactId>parent</artifactId>
    <version>1</version>
  </parent>
```
